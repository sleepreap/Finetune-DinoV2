# Dataset Organization Instructions

Please organize the dataset as follows. Ensure that the images are in JPG format and the labels are in PNG format.

## Directory Structure

```
data/
├── dataset/
│   ├──images/
│   │   ├── train
│   │   │   ├── img_50.jpg
│   │   │   ├── img_51.jpg
│   │   │   ├── ...
|   |   ├── val
│   │   │   ├── img_1.jpg
│   │   │   ├── img_2.jpg
│   │   │   ├── ...
│   │   ├── test
│   │   │   ├── img_100.jpg
│   │   │   ├── img_101.jpg
│   │   │   ├── ...
│   ├── labels/
│   │   ├── train
│   │   │   ├── img_50.png
│   │   │   ├── img_51.png
│   │   │   ├── ...
│   │   ├── val
│   │   │   ├── img_1.png
│   │   │   ├── img_2.png
│   │   │   ├── ...

```
